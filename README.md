# Python Tkinter Calculator

A GUI-based desktop calculator built using Python and Tkinter.

## Features
- Addition, subtraction, multiplication, and division
- Square root
- Percentage calculation
- Negative number handling
- Division by zero protection

## Technologies Used
- Python 3
- Tkinter

## How to Run
1. Clone the repository:
   git clone https://github.com/silentecho033/python-tkinter-calculator.git

2. Run the file:
   python calculator.py
